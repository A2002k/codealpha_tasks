# 🛍️ AK Store

A full-stack e-commerce platform built with the MERN Stack.

## Features

- User Authentication (JWT)
- Admin Dashboard
- Product Management
- Shopping Cart
- Wishlist
- Product Reviews & Ratings
- Coupons & Discounts
- Delivery Fee Calculation
- Email Notifications
- Inventory Management
- Order Tracking
- Revenue Dashboard
- Top Selling Products
- Responsive Design
- Export Orders to Excel

---

## Technologies

### Frontend
- React
- React Router
- Axios
- CSS
- Recharts
- XLSX

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Nodemailer
